# 機器學習與電腦視覺作品集

本專案收錄本人於機器學習、深度學習與電腦視覺領域的實作成果，使用 Python、OpenCV、Scikit-learn 與 PyTorch 等技術完成模型訓練與影像分析。

---

## 專案內容

### 1. GAN 影像生成模型（GANs）

#### 專案簡介

利用生成對抗網路（Generative Adversarial Network, GAN）進行影像生成。

#### 使用技術

* Python
* PyTorch
* Deep Learning
* GAN

#### 學習重點

* Generator 與 Discriminator 架構設計
* 模型訓練流程
* 影像生成技術

---

### 2. LeNet 手寫數字辨識（LeNet CNN）

#### 專案簡介

使用 LeNet 卷積神經網路模型進行 MNIST 手寫數字辨識。

#### 使用技術

* PyTorch
* CNN
* MNIST Dataset

#### 學習重點

* 卷積神經網路架構
* 影像分類
* 模型訓練與驗證

---

### 3. HOG + SVM 影像分類

#### 專案簡介

使用 HOG（Histogram of Oriented Gradients）擷取影像特徵，再透過 SVM 建立分類模型。

#### 使用技術

* OpenCV
* Scikit-learn
* HOG
* SVM

#### 學習重點

* 特徵工程
* 傳統機器學習方法
* 影像辨識流程

---

### 4. HOG + Neural Network

#### 專案簡介

結合 HOG 特徵與神經網路模型進行影像分類，並比較不同模型效能。

#### 使用技術

* OpenCV
* PyTorch
* HOG Feature

#### 學習重點

* 特徵萃取
* 神經網路訓練
* 模型效能分析

---

### 5. 機器學習實作

#### 專案簡介

實作多種機器學習模型，包含資料前處理、模型訓練與預測分析。

#### 使用技術

* Scikit-learn
* Pandas
* NumPy

#### 學習重點

* 資料清理
* 特徵選擇
* 分類與迴歸模型
